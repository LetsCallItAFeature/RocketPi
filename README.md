Das Hauptziel dieses Projektes ist ein auf dem Raspberry Pi zero w basierendes Gerät, das einen Countdown zum nächsten Start einer Rakete anzeigt und Informationen zu der Rakete/Mission darstellt.
Das Gerät besitzt zwei Anzeigen: ein vier Ziffern-Siebensegmentdisplay mit I2C Backpack von Adafruit für Zeitangaben wie den Countdown sowie ein 16x2 Character LCD-Display für Text.

Darüber hinaus verfügt das Gerät im vollendeten Zustand über weitere Funktionen:
 - Die Ansteuerung von mini Scheinwerfern und flackernden LEDs für die Beleuchtung einer Modell-Rakete und deren Triebwerke passend zum Start der echten
 - Wiedergabe eines Musikstückes passend zum Start
 - ein umfangreiches Einstellungsmenü - derzeit in Arbeit
 - Möglichkeit, falls verfügbar, einen Link zu dem Stream des aktuellen Starts per Knopfdruck zu verschiedenen Geräten (PC, Smartphone) zu schicken - TODO
